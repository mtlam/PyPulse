PyPulse v0.0.1¶

PyPulse is a package used to handle PSRFITS files and perform subsequent analyses on pulse profiles.

Download:

Github

Requirements:

  • python 2.7 or 3.X
  • numpy
  • scipy
  • matplotlib
  • astropy or pyfits
